# Completar qualsevol exercici (prompt per a OpenCode)

Copia i enganxa això a OpenCode:

```
Vull que facis l'Exercici N del repositori del curs.

Obre 30-code/README.md, llegeix l'índex i digues-me quins exercicis hi ha (01 a 08).
Jo et diré el número. De moment fes:

1. Llegeix el codi de l'exercici
2. Explica'm què fa pas a pas
3. Busca errors potencials o millores
4. Desa una versió millorada amb el nom Exercici-N-millorat.ino
5. Verifica que compila amb arduino-cli

Si l'exercici fa servir MQTT, pregunta'm la IP del broker primer.
Si fa servir WiFi, pregunta'm el SSID i password.

FQBN per ESP32: esp32:esp32:esp32-s3
```
