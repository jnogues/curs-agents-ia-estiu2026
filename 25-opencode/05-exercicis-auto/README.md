# OpenCode fa exercicis sol

Quan ja sàpigues usar OpenCode, deixa que ell faci la feina per tu.

Obre el TUI dins de la carpeta de l'exercici:

```bash
cd 30-code/Exercici-01/
opencode
```

| Prompt | Què fa |
|--------|--------|
| [Completar exercici 01](completar-ex01.md) | Llegeix, analitza i millora l'Exercici 01 del curs |
| [Completar qualsevol](completar-qualsevol.md) | Tria l'exercici (01-08) que vulguis fer |
| [Depurar codi](depurar-codi.md) | Revisa i corregeix errors automàticament |
