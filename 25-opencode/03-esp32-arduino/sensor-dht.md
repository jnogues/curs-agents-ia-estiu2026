# Sensor DHT11/22 (ESP32) — prompt per a OpenCode

Copia i enganxa això a OpenCode:

```
Genera un sketch d'Arduino per a ESP32 que llegeixi un sensor DHT11 (o DHT22) connectat al GPIO4 i mostri temperatura i humitat pel port sèrie cada 2 segons.

- Framework: Arduino
- Utilitza la llibreria DHT sensor library d'Adafruit
- Port sèrie a 115200 bauds
- Desa'l com a dht11.ino, compila'l i puja'l

FQBN: esp32:esp32:esp32-s3 o esp32:esp32:esp32
```
