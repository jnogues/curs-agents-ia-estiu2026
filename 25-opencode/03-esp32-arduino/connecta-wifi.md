# Connectar-se a WiFi (ESP32) — prompt per a OpenCode

Copia i enganxa això a OpenCode:

```
Genera un sketch d'Arduino per a ESP32 que:

1. Es connecti a una xarxa WiFi
2. Faci una petició HTTP GET a http://example.com
3. Mostri la resposta pel port sèrie (115200 bauds)

Fes que el SSID i password siguin variables editables al principi del codi.
Desa'l com a wifi-test.ino, compila'l i puja'l al port que detectis.

FQBN: esp32:esp32:esp32-s3 o esp32:esp32:esp32
```
