# Programar ESP32 amb Arduino

Guies per programar plaques **ESP32** amb **Arduino framework** usant OpenCode.

Abans de començar, obre OpenCode en mode TUI dins de la carpeta del teu projecte:

```bash
mkdir ElMeuProjecte
cd ElMeuProjecte
opencode
```

Dins del TUI, demana-li el que necessitis:

| Prompt | Què fa |
|--------|--------|
| [Blink](blink.md) | LED parpellejant al GPIO2 |
| [Connectar WiFi](connecta-wifi.md) | Connectar-se a WiFi i fer petició HTTP |
| [Sensor DHT](sensor-dht.md) | Llegir temperatura i humitat |
| [MQTT](mqtt.md) | Publicar i subscriure's a MQTT |
