# MicroPython per ESP8266

Guies per programar la **NodeMCU** amb **MicroPython** enlloc d'Arduino.

Obre el TUI d'OpenCode dins d'una carpeta de projecte i demana-li el que necessitis:

```bash
mkdir ProjecteMicroPython
cd ProjecteMicroPython
opencode
```

| Prompt | Què fa |
|--------|--------|
| [Flashejar MicroPython](flasheja-micropython.md) | Esborra flash, descarrega firmware i flasheja |
| [Pujar scripts](puja-scripts.md) | Puja boot.py i exercicis al dispositiu |

> Els exercicis de MicroPython els trobareu a [../35-micropython-esp8266/](../35-micropython-esp8266/).
