# Sensor DHT11/22 (ESP8266) — prompt per a OpenCode

Copia i enganxa això a OpenCode:

```
Genera un sketch d'Arduino per a ESP8266 (NodeMCU V2) que llegeixi un sensor DHT11 (o DHT22) i mostri temperatura i humitat pel port sèrie cada 2 segons.

- Connecta el DHT a un GPIO lliure (ex: GPIO14, però adapta'l si cal)
- Framework: Arduino
- Utilitza la llibreria DHT sensor library d'Adafruit
- Port sèrie a 115200 bauds
- Desa'l com a dht11.ino, compila'l i puja'l
- FQBN: esp8266:esp8266:nodemcuv2

💡 Pinout de referència: hardware.md
```
