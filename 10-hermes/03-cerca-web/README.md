# 🌤️ Activitat 03: Cerca web

**Durada:** ~10 minuts

## Objectiu

Descobrir que Hermes pot navegar per Internet i buscar informació real!

## Passos

### 1. Connecta't i obre Hermes

```bash
ssh userXX@178.105.91.211
export PATH="$HOME/.local/bin:$PATH"
hermes
```

### 2. Pregunta el temps

> *"Quin temps fa avui a Barcelona?"*

Hermes buscarà la informació a Internet i et donarà la previsió actual.

### 3. Busca notícies

> *"Busca'm 3 notícies recents sobre intel·ligència artificial"*

### 4. Fes una cerca més específica

> *"Quina és la població de Cerdanyola del Vallès?"*

> *"Qui va guanyar la Champions League l'any 2024?"*

### 5. Demana informació tècnica

> *"Quines són les últimes versions de Python i Node.js?"*

## ✨ Què has après?

- Que Hermes pot consultar Internet en temps real
- Que distingeix entre "parlar de memòria" i "buscar ara"
- Que pots usar-lo per informar-te de coses actuals

## 💡 Per anar més lluny

- Demana-li que **comparin dos articles** o fonts d'informació
- Pregunta-li per un tema del qual no en sàpigues res i que t'ho expliqui
- Prova: *"Cerca el preu actual del Bitcoin i explica'm per què ha pujat o baixat"*

## ❌ No funciona?

- Si diu "no puc navegar per Internet", prova de ser més explícit: *"Busca a Internet..."*
- Si es queixa de permisos, avisa el profe
- La cerca pot trigar uns segons — és normal!
