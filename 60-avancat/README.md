# 🚀 Mòdul avançat: Claude Code + ESP-IDF

> Per alumnes que volen anar més enllà de l'Arduino IDE i programar l'ESP32 com un professional.

Benvingut/da al mòdul avançat! Aquí aprendràs a programar l'ESP32 amb l'**ESP-IDF** (Espressif IoT Development Framework), el SDK oficial d'Espressif, combinat amb **Claude Code** com a assistent d'IA.

## Què és ESP-IDF?

L'ESP-IDF és el framework de desenvolupament oficial d'Espressif per als seus xips ESP32, ESP32-S3, ESP32-C3 i més. A diferència de l'Arduino IDE:

| Arduino IDE | ESP-IDF |
|-------------|---------|
| 🧒 Fàcil per començar | ⚡ Accés total al xip |
| 📚 Llibreries simplificades | 🔧 Configuració fins a nivell de registre |
| 🐌 Rendiment limitat | 🚀 Optimització professional |
| 🔒 Funcions avançades limitades | 🔓 FreeRTOS, memòria dinàmica, perifèrics avançats |

Per què servirà Claude Code aquí? Perquè l'ESP-IDF és _massiu_ — centenars de fitxers d'include, configs Kconfig, makefiles, i documentació extensa. Claude Code pot generar codi ESP-IDF des de zero, configurar el sistema de builds i depurar errors de compilació.
